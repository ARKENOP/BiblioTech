Ajouter un livre

Le titre est requis. Le titre doit comporter au moins 3 caractères.
L'auteur est requis. Le nom de l'auteur doit comporter au moins 3 caractères.
La description est requise. La description doit comporter au moins 10 caractères.
La catégorie est requise. La catégorie doit comporter au moins 3 caractères.
